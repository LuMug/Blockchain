Forger

Forge transactions and create wallets

                        ./forger [-priv <path>]
                                 [-out <path>]
                                 [-amount <value>]
                                 [-to <address>]
                                 [-dump <path>]
                                 [-last <path>]
                                 [-gen]
                                 [-help]
                                 [-examples]
                                 [-first]



Seeder

Peer discovery server

                        ./seeder <port>



Full Node

Full functional node

                        ./node [-p <port>]
                               [-db <file>]
                               [-help]



Miner Node

Full node with mining capabilities

                        ./miner -priv <file>
                                [-p <port>]
                                [-db <file>]
                                [-cores <number>]
                                [-help]



Api Node

Full node with POST HTTP routes

                        ./api [-nodeport <port>]
                              [-apiport <port>]
                              [-db <file>]
                              [-keystore <keystore.jks>
                               -password <password>]
                              [-help]



Webserver 

                        ./webserver -www <path>
                            [-port <port>]
                            [-db <file>]
                            [-ssl <keystore.jks> <password>]
                            [-help]

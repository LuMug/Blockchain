A blockchain from scratch

A blockchain built in Java from scratch, with a custom language to deploy smart contracts.


Executables

  • Forger - Generate wallets and sign transactions
  • Seeder - Peer discovery server
  • Full Node - Full node
  • Miner Node - Node mining block
  • Api Node - HTTP API block
  • Webserver - Webserver for this website
